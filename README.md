# Mental Health Questionnaire Application
A Java-based desktop application designed to provide users with a preliminary mental health assessment through structured questionnaires.

## Key Features
* **Dynamic Scoring:** Implements logic to calculate results based on user responses.
* **OOP Architecture:** Built using Object-Oriented principles for scalability and maintainability.
* **Input Validation:** Ensures all questions are answered before generating a report.

## Tech Stack
* **Language:** Java (JDK 17+)
* **UI:** Swing / JavaFX (whichever you used)
* **Storage:** File-based / Local Database (whichever you used)

## How to Run
1. Clone the repository.
2. Open in IntelliJ IDEA, Eclipse, or NetBeans.
3. Run `Main.java`.
